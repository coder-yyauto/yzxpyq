# 校园社交平台项目接管计划

## 项目概述
这是一个模仿微信朋友圈的校园社交平台，支持教师和学生用户类型，提供动态发布、评论回复、点赞等功能。

- **前端技术**：Vue.js + ElementUI
- **后端技术**：Flask + SQLAlchemy
- **数据库**：SQLite

## 接管步骤

### 1. 环境搭建与依赖安装
- 安装后端依赖：`cd backend && pip install -r requirements.txt`
- 安装前端依赖：`cd frontend && npm install`
- 初始化数据库：`cd backend && python init_db.py`

### 2. 项目启动与验证
- 启动后端服务：`./start_backend.sh`
- 启动前端服务：`cd frontend && npm run serve`
- 访问应用：http://localhost:8080
- 验证核心功能：
  - 用户注册登录
  - 动态发布
  - 图片上传
  - 点赞评论
  - 管理功能

### 3. 代码结构分析
- **后端结构**：
  - `app.py`：主应用文件，包含所有API接口
  - `init_db.py`：数据库初始化
  - `uploads/`：图片上传目录
  - `instance/`：数据库文件目录
- **前端结构**：
  - `src/views/`：页面组件
  - `src/api/`：API调用
  - `src/store/`：状态管理
  - `src/router/`：路由配置

### 4. 核心功能测试
- **用户系统**：注册、登录、权限验证
- **动态功能**：发布、查看、删除

- **管理功能**：用户管理、系统配置
- **系统功能**：备份、恢复、清空
